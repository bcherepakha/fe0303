# Изучаем git

## 1. Знакомство с Git Commit

- [x] Коммиты в GIT:

```sh
git commit
git commit
```

- [X] Ветвление в Git:

```sh
git branch BugFix
git checkout BugFix
```

- [X] Ветки и слияния:

```sh
git branch BugFix
git checkout BugFix
git commit
git checkout master
git commit
git merge BugFix

```

- [X] Git Rebase

```sh
git branch BugFix
git checkout BugFix
git commit
git checkout master
git commit
git checkout BugFix
git rebase master
```
